# 🚀 FLABLE.AI - Quick Start

## ⚡ ONE-CLICK START

Just **double-click** this file:
```
START.bat
```

That's it! 🎉

---

## What It Does

1. ✅ Checks if setup is needed
2. ✅ Installs dependencies (first time only)
3. ✅ Creates database
4. ✅ Starts backend (port 8000)
5. ✅ Starts frontend (port 3000)
6. ✅ Opens browser automatically

---

## First Time Setup

**Double-click `START.bat`**

It will:
- Create Python virtual environment
- Install all Python packages
- Install all Node.js packages
- Create SQLite database
- Start everything!

Takes ~3-5 minutes first time.

---

## Every Time After

**Double-click `START.bat`**

Starts in ~10 seconds!

---

## URLs

- **App**: http://localhost:3000
- **API**: http://localhost:8000
- **Docs**: http://localhost:8000/docs

---

## Stopping

Press any key in the command window.

---

## Shopify Setup (Optional)

1. Edit `backend\.env`
2. Add your credentials:
   ```env
   SHOPIFY_CLIENT_ID=your_id
   SHOPIFY_CLIENT_SECRET=your_secret
   ```
3. Restart `START.bat`
